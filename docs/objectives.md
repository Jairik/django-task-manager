# Take-Home Assessment

For this take-home assessment, you will build a very basic task management application.

You may use AI, but you will still be expected to defend your design and implementation decisions in a follow-up technical interview.

We expect this assessment to take most candidates **2–4 hours**. Please do not spend more than **6 hours** on it.

## Tech Stack

For this project, you will use:

- **Python**
- **Django**
- A relational database of your choice
- Django templates
- A UI library of your choice, such as:
  - Tailwind CSS
  - Bootstrap

## Project Requirements

You will build a basic task management system with the following functionality:

1. Create a project with fields such as:
   - Project name
   - Due date
   - Priority
   - Description

2. Create a task associated with a project, with fields such as:
   - Task name
   - Associated project
   - Priority
   - Due date
   - Status

3. View all projects with a due-date filter.

4. View all tasks associated with a project.

5. Edit a project.

6. Edit a task.

7. Delete a project.

8. Delete a task.

9. Include at least one automated test.

10. Include a `README.md` file with instructions for running the project.

## User Management

You do not need to build any user-management or login features.

You may assume that only one person will use the system.

## Handling Ambiguity

Part of this assessment is intended to evaluate your ability to handle ambiguity.

When something is unclear, make the decision that you believe makes the most sense. Your decisions can be discussed during the follow-up interview.

## Submission Instructions

Push your completed project to a **public GitHub repository** and share the repository link with the interview team.

## Discussion Preparation

During the follow-up interview, you will be expected to discuss:

- Your design choices
- Your database schema
- Your technology-stack decisions

You will also be asked to add a small additional feature live, without using AI, to demonstrate your understanding of the codebase.
